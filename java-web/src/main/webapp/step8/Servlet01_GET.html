GET 요청의 예

0. 웹브라우저에 URL 입력

"http://localhost:9999/java-web/step8/Servlet01" 과 같은 URL 주소를 웹브라우저에 입력하면 웹 브라우저는 무조건 GET요청을 보낸다

1. 링크

링크를 누르면 웹 브라우저는 서버에 GET 요청을 보낸다.

2. 폼(method 속성 생략)

form 태그는 입력 양식을 만들때 주로 사용한다. form 태그에 method 속성을 생략하면 기본이 GET이다
name:
age :

2. 폼(method 속성 추가)

form 태그를 사용하여 GET요청을 할떄 method 속성의 값을 GET으로 명시해도 된다.
name:
age :

4. 자바스크립트: location.href="url"

자바스크립트의 location 객체를 사용하면 GET 요청을 할 수 있다.
