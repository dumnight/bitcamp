# 자바 프로젝트
## ver 04
- 배열 문법 응용

여러 명의 성적을 저장하고 다음과 같이 출력하라!

출력내용:
```
홍길동, 100,  90,  80, 270,  90.0
임꺽정,  80,  80,  80, 240,  80.0
유관순, 100, 100, 100, 300, 100.0
```
## ver 03
- 변수 응용 

변수를 이용하여 이름, 국어, 영어, 수학, 총점, 합계를 저장하라!그런 후 다음과 같이 출력하라!

출력 내용:
```
이름: 홍길동
국어: 100
영어: 100
수학: 100
총점: 300
평균: 100.0
```
