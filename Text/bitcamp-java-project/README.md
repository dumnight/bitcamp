# 자바 프로젝트
## ver 03
변수를 이용하여 이름, 국어, 영어, 수학, 총점, 합계를 저장하라!그런 후 다음과 같이 출력하라!

출력 내용:
```
이름: 홍길동
국어: 100
영어: 100
수학: 100
총점: 300
평균: 100.0
```
